# Wing Commander Series DREAMMpack
This pack adds games in the Wing Commander series to the DREAMM emulator.

The following games in the series are included. Not all of the games work yet.

# Wing Commander
The graphics files on floppy disk versions are all for VGA mode. The user can choose EGA or Tandy graphics instead, and the installer then converts the VGA files during installation. Most of the versions on CD include only VGA graphics, and there is no installation required - DREAMM can simply add the necessary files direct from the CD.
The Kilrathi Saga is a five-CD compilation for Windows that includes Wing Commander, Wing Commander II, and Wing Commander III. The first two games are both included on Disc 1.
